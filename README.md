# Nuxt 3 starter

A minimal starter project for static Nuxt 3 projects.

## Features:

- Vue 3 + Nuxt 3
- Tailwind CSS + Headless UI
- VueUse composables
- Icons
- ESLint setup by @antfu

## How to use:

- [ ] Clone this repo (`npx tiged Flesky/nuxt-starter`)
- [ ] Add `name` and `description` to `package.json`
